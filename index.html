  • TBS Template

    Hello World Example is using a TinyButStrong PHP Template Engine to Merge hello.php and hello.html.
    But now we going to build a pure TBS template.
     
    tbs.php <-- Click to copy code
    <?php

	// include template engine
	include_once('./includes/template/tbs_class.php');
	
	// New template engine object
	$TBS =& new clsTinyButStrong;
	
	// Load Your HTML file
	$TBS->LoadTemplate('tbs.html');    // <----- Place this script HTML
	
	// **************  Write Your Code Here  **************
	
	$message = "PHP Template Engine - TinyButStrong";
	
	$a = 1345;
	$b = 2243;
	$sum = $a + $b;

	// ****************************************************

	// Show result
	$TBS->Show() ;
	
?>
